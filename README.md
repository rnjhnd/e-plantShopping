# e-plantShopping

## Project Overview
Welcome to the **e-plantShopping** repository. This web application is designed to simulate an online store for plant enthusiasts. It allows users to browse through a catalog of houseplants, view product details, and manage a shopping cart for their purchases.

## Key Features
* **Product Listing:** A visual display of various plants available for purchase.
* **Shopping Cart:** Users can add items, adjust quantities, and remove items from their cart.
* **Checkout:** A summary page to review the order.

## Technologies Used
* JavaScript (React)
* HTML / CSS
* Node.js

## How to Run
1. Clone the repository.
2. Install dependencies using `npm install`.
3. Start the application using `npm start`.
